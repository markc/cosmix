# 2026-03-06 — Cosmix Project Created

## Session 1: Strategic Direction

### Context

Conversation in `~/.mc/` workspace explored Rust web development, scripting languages to complement Rust, and the observation that COSMIC desktop creates a unique platform opportunity.

### Key Decisions

1. **Cosmix umbrella project** — unifies appmesh, nodemesh, markweb, and cosmic under one strategic direction

2. **Lua as ARexx replacement** — Lua embedded via mlua becomes the universal scripting language for desktop automation, replacing PHP in appmesh and bash for non-interactive scripting

3. **Two-world architecture** — Desktop world (Rust+Lua on COSMIC) and Web world (Laravel+React) connected by meshd WebSocket. Clean boundary, neither tries to be the other.

4. **Neovim model** — Lua is the primary development language, Rust provides the engine. Normal development means writing Lua, not compiling Rust.

5. **Language policy refined:**
   - Rust: daemons, desktop, performance
   - Lua: scripting, automation, plugins, glue
   - PHP+React: web only (markweb)
   - bash: interactive shell only
   - Python: never
